## 🔧 使用前準備

### 必要步驟

1. **加入 VisionMaster SDK 引用**
   需要手動加入以下 DLL：
   - VM.Core.dll
   - VM.PlatformSDKCS.dll
   - VMControls.Winform.Release.dll
   - IMVSEdgeWidthFindModuCs.dll

2. **準備 solution.solw**
   - 建立視覺檢測方案
   - 至少包含一個 Procedure
   - 必須有 "間距檢測1" 工具
   - Height 輸出格式: "狀態;像素值"

3. **修改 Config.ini**
   - 設定正確的 PLC IP/Port
   - 調整 PixelToMM 比例
   - 修改登入帳密

### 編譯步驟

1. 用 Visual Studio 開啟 Vision_Master.csproj
2. 加入 VisionMaster SDK 引用
3. 建置專案
4. 將 Config.ini 和 solution.solw 放到執行檔目錄
5. 執行 Vision_Master.exe


## 📧 注意事項

1. **Config.ini 必須存在**，否則程式無法啟動
2. **預設帳密**: admin / 123456
3. **PixelToMM 需校正**：根據實際相機設定調整
4. **檢測結果格式固定**："OK;150.5" 或 "NG;145.2"
